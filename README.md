# Ecommerce Sales Dashboard

## Project Description
This project is an interactive Ecommerce Sales Dashboard created using Microsoft Excel to analyze sales performance, profit trends,
customer behavior, and category-wise business insights.
The dashboard helps in understanding key business metrics through charts, KPIs, and visual reports for better business decision-making.

## Key Features
- Sales Analysis
- Profit Analysis
- Category-wise Performance
- Order Quantity Analysis
- Region-wise Trends
- Interactive Dashboard with Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Cleaning
- Data Analysis

## Conclusion
This dashboard provides clear business insights by visualizing important Ecommerce KPIs in a simple and interactive format.
